# vue-admin-scaffold
> vue3.0 + typescript + webpack4 + element-plus 脚手架



## 浏览器兼容性
> chrome、safari、Edge

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Run your end-to-end tests
```
npm run test:e2e
```

### Lints and fixes files
```
npm run lint
```
